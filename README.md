# Shahzoda
​Shahzoda AI project for lip-sync and face animation
Shahzoda AI — bu sun'iy intellekt yordamida yuz animatsiyasi va lip-sync (lab harakati) yaratishga mo‘ljallangan loyiha.
## Loyiha maqsadlari
* Statik suratlarni yuqori sifatli video formatiga o'tkazish.
* Audio treklar yordamida yuz harakatlarini va lab sinxronizatsiyasini (lip-sync) avtomatlashtirish.

## Kelajakdagi rejalar
* Loyihani kengaytirish va yangi sun'iy intellekt modellarini integratsiya qilish.
* Jarayonni tezlashtirish va foydalanuvchilar uchun qulay interfeys yaratish.
